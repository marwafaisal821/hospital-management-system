🏥 نظام إدارة المستشفيات (HMS)
نظام شامل لإدارة المستشفيات مبني على Odoo 17، يتضمن وحدات مخصصة لإدارة المرضى، الأقسام، والأطباء، مع تكامل مع وحدات إضافية لتعزيز الوظائف.
